gitbash
pwd : 현재 위치 확인
Desktop/ : 바탕화면으로 이동
mkdir 폴더명 : 폴더 생성
ls : 현재 위치의 파일/폴더 목록 출력
git init : 현재 위치를 working directory 로 초기화
ㄴ 폴더의 숨긴항목 보기 > .git 폴더가 있음
	ㄴ 해당 폴더를 지우면, working directory 제외
wd 안에서 변화가 생기면, 해당 내용을 git이 추척할 수 있다.
ㄴ 기록 후 수정되지 않은 파일
    기록 후 수정 또는 생성된 파일
git add ./파일명 : 해당 파일의 수정사항을 staging area에 기록(등록)
ㄴ git add . : 현재 폴더의 모든 수정사항들을 등록
ㄴ staging area : 기록으로 남길 내용을 임시저장(등록)하는 공간(장바구니)
git commit -m '메시지' : local repository에 [ 메시지 + staging area의 기록 ] 등록 및 staging area 기록 삭제
ㄴ local repository : 기록으로 남길 내용을 최종저장하는 공간

git config --global user.name "이름" : git에 사용자명 등록
git config --global user.email "이메일" : git에 이메일 등록
git config --system --unset credential.helper : 리셋
git config --list : git에 등록된 리스트 출력
git log : local repository에 남겨진 기록(메시지) 보기
